Webhook Guide

The image is an example of webhook. You can use a webhook to post embed announcements for your server and easily hide long links with plain text.

To create a webhook, go to Server Settings > Integrations > Webhooks > New Webhook
Choose the desired channel, webhook name, and webhook image. Now, click on Copy Webhook URL, and paste it in the Webhook URL field above.

Tips:

  • Leave the field empty if you do not need the field.
  • To use a hyperlink use [Cartero](https://cartero.cf/webhook)
  • The default embed colour is black.
  • To tag everyone type @here or @everyone in Plain Content field.
